Property Management API Documentation

Backend API for managing property listings

Base URL

https://jess-unbroadcasted-audrina.ngrok-free.dev/api/properties

If using ngrok, replace with your public URL.

Add Property (Admin)

POST /api/admin/property

Request Body

FieldTypeDescription
titlestringProperty title
descriptionstringDetailed description of property
locationstringCity or area name
latitudenumberLatitude coordinate
longitudenumberLongitude coordinate
bhknumberBHK configuration
property_typestringApartment, Villa, Plot etc.
construction_typestringNew or Resale
construction_statusstringReady to Move / Under Construction
pricenumberProperty price
area_sqftnumberTotal area in square feet
verifiedbooleanWhether property is verified

Example Request

{
"title":"Luxury 3BHK Apartment",
"description":"Beautiful apartment with city view",
"location":"Jaipur",
"latitude":26.9124,
"longitude":75.7873,
"bhk":3,
"property_type":"Apartment",
"construction_type":"New",
"construction_status":"Ready to Move",
"price":7500000,
"area_sqft":1450,
"verified":true
}

Example Response

{
"message": "Property added"
}

Get Dropdown Options

GET /api/options

Description

Returns unique values for dropdown filters like BHK, property type and construction status.

Example Response

{
"bhk":[{"bhk":1},{"bhk":2},{"bhk":3}],
"property_type":[{"property_type":"Apartment"},{"property_type":"Villa"}],
"construction_type":[{"construction_type":"New"},{"construction_type":"Resale"}],
"construction_status":[{"construction_status":"Ready to Move"},{"construction_status":"Under Construction"}]
}

Filter Properties

GET /api/properties

Query Parameters

ParameterDescription
cityFilter by city/location
bhkFilter by BHK
minPriceMinimum price
maxPriceMaximum price
typeProperty type

Example Request

/api/properties?city=Jaipur&bhk=3&minPrice=5000000&maxPrice=10000000&type=Apartment

Example Response

{
"data":[
{
"property_id":1,
"title":"Luxury 3BHK Apartment",
"description":"Beautiful apartment with city view",
"location":"Jaipur",

}
]
}
